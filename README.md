📻 Navidrome Radio Manager (V6.1)
🇮🇹 Descrizione (Italiano)
Navidrome Radio Manager è una web-app leggera e intuitiva basata su Streamlit, progettata per semplificare la gestione delle stazioni radio nel tuo server Navidrome. Invece di inserire manualmente URL e metadati, questa app ti permette di cercare tra migliaia di stazioni globali e aggiungerle con un solo clic.

✨ Caratteristiche Principali
Ricerca Ibrida: Cerca per nome o per nazione (utilizzando la lista ufficiale di Radio-Browser).

Controllo Duplicati: L'app interroga Navidrome in tempo reale e ti avvisa (con un'icona ✅) se una radio è già presente nel tuo database.

Multi-lingua: Supporto integrato per Italiano e Inglese tramite variabile d'ambiente.

Resilienza: Utilizza un sistema di mirror bilanciato (all.api) per garantire che la ricerca funzioni sempre.

Docker Ready: Immagine ottimizzata per un deploy rapido e pulito.

🇬🇧 Description (English)
Navidrome Radio Manager is a lightweight Streamlit-based web application designed to streamline radio station management for your Navidrome server. Instead of manually copying URLs and metadata, this app lets you browse thousands of global stations and add them with a single click.

✨ Key Features
Hybrid Search: Search by name or country (using the official Radio-Browser list).

Duplicate Check: The app queries Navidrome in real-time and warns you (with a ✅ icon) if a station is already in your database.

Multi-language: Built-in support for Italian and English via environment variables.

Resilience: Uses a balanced mirror system (all.api) to ensure the search is always available.

Docker Ready: Optimized image for fast and clean deployment.
